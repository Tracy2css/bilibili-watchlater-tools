### Script Metadata (for Tampermonkey)
To enable automatic updates, ensure your script header looks like this:

```javascript
// ==UserScript==
// @name         Bilibili Watch Later - Move Selected to Top
// @version      14.0
// @author       Tracy2css
// @match        *://[www.bilibili.com/list/watchlater](https://www.bilibili.com/list/watchlater)*
// @updateURL    [https://raw.githubusercontent.com/Tracy2css/bilibili-watchlater-tools/main/bilibili-top.user.js](https://raw.githubusercontent.com/Tracy2css/bilibili-watchlater-tools/main/bilibili-top.user.js)
// @downloadURL  [https://raw.githubusercontent.com/Tracy2css/bilibili-watchlater-tools/main/bilibili-top.user.js](https://raw.githubusercontent.com/Tracy2css/bilibili-watchlater-tools/main/bilibili-top.user.js)
// @grant        GM_xmlhttpRequest
// ==/UserScript==
